Version 1

Background

The adventure is “Storm King’s Thunder”, a new adventure for characters level 1–12. Adventurers of all experience levels are welcome. The group size will be >= 4 players.

As an adventurer, you don’t need to buy anything. However, a good set of dice will always help! [0]

Info

  • We will be using audio to communicate (webcams optional but recommended) over something like Skype or Discord.
  • We will play on https://roll20.net/, which is a virtual table top. In-person play will still be possible, though laptops will have to be shared (etc).
  • We will run a “session 0” to make sure we all know how to use roll20.net, and to introduce everyone’s characters.

Characters

Character creation is easier than it sounds. For all adventurers, I will be more than happy to spend some time with you helping you figure out what kind of character you want to play, and get everything ready with you. For newbies, this will probably take 1–2 hours, because there will be more things to explain.

If you were part of the group that played “Lost Mine of Phandelver”, you can carry on using your character. However, you can retcon/change any aspect of your characters – including ability scores, skills, race, class, background information – now that you know more about them.

For veteran adventurers who want to dive straight in:

  • Characters start at level 5.
    • Create a level 1 character, then progress through level-ups one at a time.
    • Initial ability scores are to be picked from: 15, 14, 13, 12, 10, 8.
  • Please read the “game rules” section below for restrictions.

To reiterate, I am happy to help everyone with this!

Games rules

  • Characters must not of an Evil alignment.
  • Characters must not be any of the following races: Dragonborn, Duergar, Deep Gnome, Genasi, Aasimar.
  • Half-elf racial variants and/or Tiefling racial variants (as described in the “Sword Coast Adventurer’s Guide” book) are not allowed.

Known unknowns

  • I like rolling real dice, so I am looking to see how we can do that and yet have it all work within roll20.net.
  • I am learning how to use roll20.net properly so I can teach everyone.
